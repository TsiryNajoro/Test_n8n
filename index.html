📦 GitHub Repository

This demo listens to webhook events triggered by repository updates:

  • issues.opened -> Triaged by AI
  • push -> Syncs code docs
  • pull_request -> Notifies DevOps

⚡ n8n Workflow

The operational backbone processing incoming data streams:

  • Webhook Node: Collects incoming JSON data
  • Switch Node: Routes events by payload type
  • Output Node: Dispatches updates back to GitHub
